      🔍 Sistema di Analisi Avanzato

      Whysafe esegue controlli multi-livello per rilevare siti pericolosi:

      Livello 1: Analisi Locale (istantanea)

      • HTTPS: Se il sito non usa crittografia (−40%)
      • Indirizzo IP: Uso di IP invece di dominio (−35%)
      • Punycode/IDN: Possibile spoofing di dominio (−20%)
      • TLD sospetti: Domini con estensioni rischiose come .tk, .ml (−15%)
      • URL lungo: Possibile obfuscation (−18%)
      • Caratteri @ e null bytes: Tecniche di phishing note (−25%, −20%)
      • Typosquatting: Molti trattini nel dominio (−7 a −15%)
      • URL Abbreviati: Nascondono il vero dominio (−15%)

      Livello 2: Verifiche Esterne (gratuita)

      • URLhaus: Database aggiornato di siti malevoli rilevati da ricercatori di malware (−50% se trovato)
      • Google Safe Browsing: Database globale di phishing, malware, software indesiderato (−60% se rilevato) — richiede configurazione API

      Livello 3: Rilevamento Contenuto Illegale

      Whysafe ora identifica automaticamente siti che ospitano contenuto illegale:

      • 🎬 Torrent Illegali: The Pirate Bay, 1337x, RARBG, Kickass, etc.
      • 📺 Streaming Pirata: FMovies, SolarMovies, Putlocker, 123Movies, Soap2Day, etc.
      • 📥 File Sharing Illegale: RapidGator, Uploaded.net, Megashare, oBoom, etc.
      • 📚 Ebook Pirata: Z-Library, BookZZ, Library Genesis, SciHub, etc.
      • ⚠️ Pattern URL Sospetti: Analizza keywords e struttura URL per rilevare siti illegali

      Risultato: Se rileva contenuto illegale, il punteggio scende drasticamente a 5% con avvertenza critica. Questi siti espongono l'utente a malware, spyware, e virus.

      Livello 4: Indice di Affidabilità del Sito (0-100%)

      Un punteggio dedicato che calcola quanto è affidabile il dominio stesso, indipendentemente dal contenuto:

      • Verifica Esistenza Sito: Controlla se il sito è raggiungibile e risponde correttamente
      • Popolarità del Sito: Siti noti e affidabili (Google, Wikipedia, GitHub) hanno score maggiore
      • Storico di Sicurezza: Se il sito ha avuto problemi di malware/phishing in passato (VirusTotal, URLhaus storico)
      • Reputazione del TLD: Domini con TLD sospetti (.tk, .ml) hanno score inferiore
      • Pattern di Phishing: Domini che contengono parole tipiche di phishing (login, verify, account)
      • Complessità Dominio: Troppi sottodomini o struttura strana riduce la fiducia

      Scoring Finale

      • 80-100%: ✓ SICURO — Probabilmente legittimo
      • 60-79%: ⚠ SOSPETTO — Verificare con attenzione
      • 30-59%: ⛔ RISCHIOSO — Evitare se possibile
      • 0-29%: 🚫 PERICOLOSO — Non visitare

      Indice di Affidabilità del Sito

      Visualizzato separatamente nella sezione risultati

      • 85-100%: ✓ ECCELLENTE — Sito molto affidabile e rinomato
      • 70-84%: ⚠ BUONO — Sito affidabile senza problemi noti
      • 50-69%: ⛔ MEDIO — Sito generico con rischi moderati
      • 0-49%: 🚫 SCARSO — Sito con problemi di affidabilità

      Come viene Giustificato il Punteggio

      Ogni indice di affidabilità mostra i "Motivi del Punteggio" — una lista dettagliata di perché il sito ha ricevuto quel punteggio specifico:

      • Verifica Raggiungibilità: Il sito risponde correttamente ai server web e è attivo
      • Dominio Conosciuto: Se il sito è nel database di domini affidabili globali
      • Reputazione TLD: Se il dominio usa un'estensione trustworthy (.com, .org) o rischiosa (.tk, .ml)
      • Storico Malware: Se il sito è mai stato rilevato con malware dai principali antivirus
      • Crittografia SSL: Se la comunicazione è cifrata con HTTPS
      • Pattern Phishing: Se il dominio contiene parole sospette come "verify", "login", "secure"
