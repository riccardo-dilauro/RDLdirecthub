    • Informativa Privacy

    Questa informativa descrive come Whysafe tratta i dati personali durante l'uso del sito e dello strumento di analisi URL. Whysafe e' progettato con approccio privacy-by-design: i controlli principali avvengono nel browser dell'utente.

    1. Titolare del trattamento

    Titolare del trattamento: Team Whysafe. Per richieste privacy puoi contattare: Whysafegroup@outlook.com.

    2. Dati trattati

    • Dati inseriti manualmente: URL digitati nel campo di controllo.
    • Dati tecnici locali: preferenza tema, chiavi API opzionali (Google Safe Browsing, VirusTotal) e cache locale delle analisi.
    • Dati di navigazione standard: log tecnici eventualmente generati dal browser o dal provider di hosting statico.

    3. Finalita e base giuridica

    • Erogazione del servizio: analisi sicurezza URL e sito web richieste dall'utente.
    • Funzionamento tecnico: memorizzazione impostazioni locali e ottimizzazione tempi di risposta.
    • Sicurezza: utilizzo di servizi esterni anti-malware/anti-phishing quando configurati.

    La base giuridica e' l'esecuzione del servizio richiesto dall'utente e il legittimo interesse a garantire sicurezza e affidabilita' operativa.

    4. Comunicazione a terze parti

    Whysafe non vende dati personali e non effettua profilazione pubblicitaria. Se abiliti API esterne, l'URL analizzato puo' essere trasmesso ai relativi provider per la verifica di sicurezza:

    • URLhaus
    • Google Safe Browsing (se API configurata)
    • VirusTotal (se API configurata)

    Le politiche privacy dei provider esterni si applicano ai dati ricevuti da tali servizi.

    5. Conservazione dei dati

    • Nel browser: impostazioni e cache sono conservate localmente fino a cancellazione manuale o reset browser.
    • Sui sistemi Whysafe: non viene mantenuto un archivio centralizzato degli URL controllati tramite interfaccia client-side.

    6. Cookie e tecnologie simili

    Il sito non utilizza cookie di marketing o tracciamento cross-site. Viene utilizzato il localStorage per funzioni tecniche (tema, chiavi API opzionali, cache).

    7. Misure di sicurezza

    • Architettura prevalentemente client-side per ridurre trasferimento dati.
    • Timeout e validazione per chiamate API esterne.
    • Nessuna esposizione intenzionale delle chiavi API oltre al browser locale dell'utente.

    8. Diritti dell'interessato

    Hai il diritto di chiedere informazioni su eventuali trattamenti riferibili alla tua persona, nonche' di richiedere aggiornamento, limitazione o cancellazione dei dati eventualmente trattati. Per dati locali puoi cancellare direttamente storage e cache del browser.

    9. Trasferimenti extra UE

    L'uso di provider esterni puo' comportare trasferimenti internazionali di dati tecnici (es. URL analizzati). Ti invitiamo a consultare le informative dei singoli provider per dettagli su localizzazione e garanzie applicate.

    10. Aggiornamenti dell'informativa

    Questa informativa puo' essere aggiornata in caso di modifiche tecniche o normative. La versione pubblicata sul sito e' quella attualmente in vigore.

    Nota: questa informativa e' redatta con finalita' informative generali e non sostituisce consulenza legale professionale.
